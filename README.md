# 📄 PDF 极客工具箱 (PDF Geek Toolbox)

一个基于 **Vue 3 + Vite + TypeScript + Tailwind CSS** 构建的轻量级、纯前端 PDF 处理工具。

### ✨ 核心特性

- **🔒 100% 隐私安全**：所有 PDF 处理（拆分、合并、打包）均在浏览器本地的沙盒环境中完成，文件绝对不会上传到任何服务器，完美保护您的隐私和数据安全。
- **📄 智能拆分**：
  - 自动解析 PDF 总页数。
  - 支持“连续区间选择”与“高级自定义输入”（如 `1-3, 5, 8-10`）。
  - 支持将拆分出的页面【合并为一个新 PDF】或【按单页切片并打包为 ZIP 压缩包】下载。
- **🔗 顺滑合并**：
  - 支持多文件批量上传与追加。
  - 列表实时显示每个文件的页数及预估合并总页数。
  - 集成原生的 **HTML5 拖拽排序 (Drag and Drop)**，可自由调整合并文件的先后顺序。
